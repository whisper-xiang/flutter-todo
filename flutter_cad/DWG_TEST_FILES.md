# DWG 测试文件

## 示例 DWG 文件下载

### 免费示例文件
1. **AutoCAD 示例文件**
   - [Architectural Sample](https://help.autodesk.com/view/ACD/2023/ENU/?guid=GUID-A5A4E8B6-9B0C-4A0E-B9A6-9E0C9B9A0A0A)
   - [Mechanical Sample](https://help.autodesk.com/view/ACD/2023/ENU/?guid=GUID-B5B4E8B6-9B0C-4A0E-B9A6-9E0C9B9A0A0A)

2. **Open Design Alliance 示例**
   - 访问 ODA 官网获取测试文件
   - 包含各种 DWG 版本 (R14 到 2024)

### 本地测试文件路径
```
assets/test_files/
├── sample_architectural.dwg
├── sample_mechanical.dwg
├── simple_2d.dwg
└── complex_3d.dwg
```

### 文件权限设置
确保测试文件具有读取权限：
```bash
chmod 644 assets/test_files/*.dwg
```
